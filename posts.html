CATEGORIAS

Selecione uma das categorias abaixo pra ler seus devidos posts.

Front-End
O desenvolvimento Front-End da web é o desenvolvimento da interface gráfica do usuário de um site, por meio do uso de linguagens como HTML, CSS e JavaScript, para que os usuários possam visualizar e interagir com o site. Ver posts sobre Front-End.
Back-End
Back-end é toda a parte da programação voltada ao funcionamento interno de um software. É tudo aquilo que está por trás da interface de uma aplicação: seus sistemas, banco de dados, segurança de dados, envio e recebimento de informações e armazenamento. Ver posts sobre Back-End.
Frameworks
Um framework em desenvolvimento de software, é uma abstração que une códigos comuns entre vários projetos de software provendo uma funcionalidade genérica. Um framework pode atingir uma funcionalidade específica, por configuração, durante a programação de uma aplicação. Ver posts sobre frameworks.
WordPress
WordPress é um sistema livre e aberto de gestão de conteúdo para internet, baseado em PHP com banco de dados MySQL, executado em um servidor interpretador, voltado principalmente para a criação de páginas eletrônicas e blogs online. Ver posts sobre WordPress.
